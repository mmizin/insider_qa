# Product Recommendations - Simple Analysis

This is a simplified overview of product recommendations on Hepsiburada for Main Page, Product Detail Page, and Cart Page.

---

## 1. Main Page

- **High-Rating Products**: Shown based on star rating and number of reviews.  
- **Popular / Bestsellers**: Trending items across the platform.  
- **Personalized Recommendations**: Based on user browsing and purchase history.

---

## 2. Product Detail Page

- **Customers Also Bought**: Products often purchased together.  
- **Similar Products**: Items with similar brand, category, price, or features.

- **High-Rating / Top-Rated**: Best-reviewed products in the same category.

---

## 3. Cart Page

- **Frequently Bought Together**: Products commonly purchased with cart items.  
- **Complementary Products**: Items that enhance the main purchase.  
- **Personalized Recommendations**: Tailored based on user history and cart contents.

---

## 4. Summary

- New users usually see popular/general items.  
- Recommendations improve with more user data.  
- Rules combine **popularity, personalization, and product similarity**.  
- Main goal: increase engagement, conversion, and average order value.
